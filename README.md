# WebGL Solar System
A computer graphics course project demonstrating a real-time 3D solar system rendered using WebGL.

## Features
- 3D planets with rotation
- Orbit simulation
- Sun light shading
- Texture support for Sun and Earth
- Written in pure JavaScript + WebGL (no game engines)
